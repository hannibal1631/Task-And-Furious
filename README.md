# Task And Furious

Task And Furious is a modern, responsive task management web application focused on simplicity, speed, and a clean user experience.

Work in progress — actively being developed.

---

## My Role (Frontend)

Leading the frontend development of the project:

* Designed and built the full user interface
* Developed core task management features
* Focused on responsiveness, performance, and usability

---

## Backend Development

The backend is being developed by a collaborating developer Riktam Nandi(https://github.com/rdxnandi) as part of this project. This collaboration is intentional, aimed at building real-world experience working in a team environment, including coordination between frontend and backend systems.

---

## Tech Stack

### Frontend

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* Axios (API communication)
* React Router (client-side routing)

### Backend

* Node.js & Express.js
* MongoDB (via Mongoose)
* JWT Authentication
* Cloudinary (media handling)
